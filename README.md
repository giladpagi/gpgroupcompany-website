# GP Group Company Website

Official website for GP Group Company, hosted on GitHub Pages.

**Live URL:** https://gpgroupcompany.com

## Development

This repository is linked as a submodule in the main monorepo.

To make changes:
1. Edit files locally
2. Commit and push to this repo
3. GitHub Pages will automatically deploy
